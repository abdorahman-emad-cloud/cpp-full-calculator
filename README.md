# 🧮 Simple C++ Command-Line Calculator

A versatile and user-friendly command-line calculator built with **C++**. This project supports basic arithmetic operations, scientific functions like square roots and powers, and integer-specific operations like modulo and factorials.

---

## 🚀 Features

The calculator provides a menu-driven interface with the following operations:
*   ➕ **Summation**: Add two numbers.
*   ➖ **Subtraction**: Find the difference between two numbers.
*   ✖️ **Product**: Multiply two numbers.
*   ➗ **Quotient**: Divide two numbers (with zero-division protection).
*   🔋 **Power**: Calculate $x^y$.
*   √️ **Square Root**: Get the root of non-negative numbers.
*   🔢 **Remainder**: Find the modulo (%) of two integers.
*   ❗ **Factorial**: Calculate the factorial of a non-negative integer.
